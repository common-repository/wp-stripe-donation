== Installation ==

Install AidWP as a regular WordPress plugin. Here are different ways to install plugin:

**Install by search plugin:**

1. In your Admin, go to menu Plugins > Add New
2. Search WP Stripe Donation
3. Click to install
4. Activate the plugin
5. Setup the proper settings in WP Stripe Donation menu

**Install by upload plugin:**

1. Download the latest version of WP Stripe Donation(.zip file)
2. In your Admin, go to menu Plugins > Add New
3. Select "Upload Plugin"
4. Click on "Choose File"
5. Select downloaded wp-stripe-donation.zip and click on "Install Now" button
6. Activate the plugin

== Frequently Asked Questions ==

= How can I Display the Donation Form? =
After activating the plugin, you will see ‚ÄúWP Stripe Donation‚Äù in WordPress Admin Dashboard.

Firstly, Go to ‚ÄúKey Settings‚Äù and add your stripe keys which you will get from your stripe account (See attached video).

Secondly, Create a page and name it as you wish like "Donation"

Finally, Put the shortcode [wp_stripe_donation] in that page and Update it.

You will see your donation form on that page.

= How can I Display Multiple Amounts? =
This is very easy.

Go to ‚ÄúGeneral Settings‚Äù and you will see an option ‚ÄúAmounts‚Äù there.

Put your amounts like ‚Äú5,10,50,100,500,100‚Äù and you will see them in the form.

= How can I Change the Currency? =
Go to ‚ÄúGeneral Settings‚Äù and you will see an option ‚ÄúCurrency‚Äù there.

Select your Currency from the drop down and save it. 

You will see it immediately in the amount section in the donation form.

= Can I use this for multiple donation items or product items? =
Yes, you can do it.

Go to ‚ÄúGeneral Settings‚Äù and you will see an option ‚ÄúDonation For Options‚Äù there.

Put your items like ‚ÄúItem One,Item Two,Item Three‚Äù and you will see them in the form drop down section.
